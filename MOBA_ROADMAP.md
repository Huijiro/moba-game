# MOBA Game Development Roadmap

This document outlines the high-level vision and features for the MOBA game. For detailed work tracking, see [GitHub Projects](https://github.com/Huijiro/moba-game/projects).

## Vision
A MOBA (Multiplayer Online Battle Arena) built with Godot 4.6 and C++ GDExtension, featuring dynamic character movement, combat, and strategic gameplay.

## Core Systems (Priority Order)

### Phase 1: Foundation
- [x] Project setup (Godot 4.6, CMake, GDExtension)
- [x] Basic character movement (Unit, InputManager, Navigation)
- [x] Isometric camera system (MOBACamera)
- [ ] Health and basic combat system

### Phase 2: Gameplay
- [ ] Attack/ability system
- [ ] Projectiles and effects
- [ ] Resource management (mana, energy)
- [ ] Enemy AI and pathfinding

### Phase 3: Polish & Features
- [ ] UI systems
- [ ] Animations and visual effects
- [ ] Sound and music
- [ ] Performance optimization

### Phase 4: Multiplayer/Advanced
- [ ] Network synchronization
- [ ] Multiple characters/heroes
- [ ] Objective systems (towers, minions)
- [ ] Game modes

## Current Status
- **Latest**: Basic movement and camera working
- **Next**: Combat system foundation
